Processor Status Word

Processor
Register Word

The Processor Status Word (PSW) is a special register in a computer's CPU that stores important information about the current state of the processor. It contains flags or status bits that reflect the outcome of operations and control the flow of execution. These flags provide critical details about arithmetic and logical operations, helping the CPU make decisions for subsequent instructions.

    Key components of the Processor Status Word
  1. Zero Flag: Set if the result of the last operation is zero.
  2. Carry Flag: Set if there is a carry-out in arithmetic operations (Example:addition overflow).
  3. Overflow Flag: Set if there is an overflow in signed arithmetic operations.
  4. Negative Flag: Set if the result of the last operation is negative (based on the sign bit).
